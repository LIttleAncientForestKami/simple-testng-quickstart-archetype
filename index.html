Generating app

Sub:

  1. archetypeVersion at some point (1.0 is current as of April 2016)

  2. groupId from eu.lafk to yours

  3. artifactId from sample-app to yours

  4. it should ask you for packaging during generation, that’s Java package.

mvn archetype:generate -DarchetypeGroupId=lafk.pl -DarchetypeArtifactId=simple-testng-quickstart -DarchetypeVersion=1.0 -DgroupId=eu.lafk -DartifactId=sample-ap

  1. mvn versions:display-property-updates to see if you need to update TestNG
